## 💬 Front-end
> 이번 프로젝트는 주특기 주차를 마치고 처음으로 프론트엔드와 백엔드로 나눠서 진행한 프로젝트 입니다.
> 우선적으로 주특기 주차에 배운 기본적인 CRUD기능의 구현에 집중하였고,
> 기능 구현을 끝낸 후 타당한 이유가 있는 새로운 기술스택 및 디자인패턴을 도입하려 노력했습니다.
